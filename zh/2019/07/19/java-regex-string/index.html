# Java通过正则表达式截取字符串中的一段子字符串

当我们有截取字符串中的某段特定子字符串的需求时，要善于利用正则表达式，而少用substring()截取字符串，因为substring()通过下标来提取固定位置的字符串，兼容性差。

比如提取一个文件名中的年月时间

    public static void main(String[] args) {
        testRegex();
    }

    public static void testRegex() {
        String fileName = "D:\\var\\log\\2020年9月测试文件.xlsx";
        String regex = "[0-9]{4}年[0-9]{1,2}月";
        List<String> list = getSubStr(fileName, regex);
        System.out.println(list.size() > 0 ? list.get(0) : "");
    }

    private static List<String> getSubStr(String fileName, String regex) {
        Pattern pattern = Pattern.compile(regex);
        Matcher matcher = pattern.matcher(fileName);
        List<String> list = new ArrayList<>();
        while (matcher.find()) {
            list.add(matcher.group());
        }
        return list;
    }

在Matcher类中常用的匹配目标的方法，如下：

  • matches:整个匹配，只有整个字符序列完全匹配成功，才返回True，否则返回False。但如果前部分匹配成功，将移动下次匹配的位置。
  • find:部分匹配，从当前位置开始匹配，找到一个匹配的子串，将移动下次匹配的位置。

注意：group()需要在find()匹配后再使用
